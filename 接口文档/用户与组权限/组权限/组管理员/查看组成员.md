# 查看组成员

## 1.接口描述

接口请求域名:groupAdmin/showMember
获取组成员
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| groupId | 是 | String | 组id |
| pageSize | 是 | Number | 页面大小 |
|  pageNum  |  是   | Number  |    页码      |
| search | 是 | String | 模糊搜索 |

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| memberList | Arrary | 成员列表 |
| pageNum  | Number | 当前访问的是第几页 |
|  pages   | Number |    总共有多少页    |
|  total   | Number |  总共有多少条数据  |
| success | int | 状态值,1成功,0失败 |
| info | string | 状态描述 |

## 4.用例

### 输入

```json
{
    "pageNum":1,
    "pageSize":15,
    "groupId":"AXDV64",
    "search":"asdasda"
}
```

### 输出

```json
{
    "memberList":[
{
            "username":"李白",
            "userId":123123,
            "type":2,
            "email":"sadasdsd@qq.com"
        }
    ],
    "pageNum":1,
    "pages":12,
    "total":125,
    "success":1,
    "info":"组删除成功"
}
```
