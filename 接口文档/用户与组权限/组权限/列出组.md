# 列出组

## 1.接口描述

接口请求域名:group/getList
获取组
请求方式:get

## 2.输入参数

无

## 3.输出参数

|  参数名称  |  类型  |         描述         |
| :-------: | :----: | :------------------: |
| groupList | Arrary | 组列表 |
| success | int | 状态值,1成功,0失败 |
| info | string | 状态描述 |

## 4.用例

### 输出

```json
{
    "groupList":[
{
            "groupName":"高三数学特训",
            "groupId":"XHZE46",
            "jsonFile":"jsonStorage/advceefeFWDCSAZ.JSON",
            "md5":"asdasdade23edwscsdcs",
            "type":2
        }
    ],
    "success":1,
    "info":"组获取成功"
}
```
