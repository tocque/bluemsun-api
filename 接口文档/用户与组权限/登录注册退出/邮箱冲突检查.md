# 判断邮箱

## 1.接口描述

接口请求域名:user/checkoutEmail
注册时判断邮箱是否已使用
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  email  |  是   | String | 邮箱 |

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | int | 状态值,1成功,0失败 |
| info | string | 状态描述 |

## 4.用例

### 输入

```json
{
    "email":"2323432@qq.com"
}
```

### 输出

```json
{
    "success":1,
    "info":"邮箱可用"
}
```
