# 修改密码

## 1.接口描述

接口请求域名:personPage/updateUsername
修改用户名
请求方式:get

## 2.输入参数

| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| username | 是 | String | 用户名 |

## 3.输出参数

| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
| success | int | 状态值,1成功,0失败 |
| info | string | 状态描述 |

## 4.用例

### 输入

```json
{
    "name":"asdasd"
}
```

### 输出

```json
{
    "success":1,
    "info":"修改成功"
}
```
